```ruby
numbers = [1, 2, 3, 4, 5]
```

```ruby
numbers.delete_at(1) # => will delete the index value 1
                     # => returns 1
                     # => numbers = [1, 3, 4, 5]
numbers.delete(1)    # => will delete the integer 1
                     # => returns 1
                     # => numbers = [2, 3, 4, 5]
```
