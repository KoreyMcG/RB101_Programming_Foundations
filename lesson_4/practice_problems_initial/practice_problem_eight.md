```ruby
arr = [1, 2, 3, 4, 5]
arr.take(2)
```

> Take returns a new array containing the first 'n' elements of self, where n
> is a non-negative integer.
>
> arr.take => [1, 2]
>
> Does not destructively modify arr.
>
> arr = [1, 2, 3, 4, 5]
>
> We can find all that out in the Array#take ruby documentation.
